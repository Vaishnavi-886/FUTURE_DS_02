# FUTURE_DS_02
Customer Retention & Churn Analysis

This repository contains a Power BI dashboard focused on analyzing customer retention and churn behavior to understand customer loyalty and identify factors contributing to customer loss.

Tools Used

Power BI Desktop

CSV / Excel data source

Key Analysis

Total customers, retained customers, and churned customers

Customer retention rate and churn rate

Churn analysis by customer segment and region

Trend analysis of customer retention over time

Files in This Repository

Customer_Retention_Churn.pbix – Power BI dashboard file

dashboard_preview.png – Dashboard screenshot

README.md – Project documentation

How to View

Download the .pbix file and open it using Power BI Desktop.
